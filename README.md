# Wine Quality Prediction using Tkinter GUI

## Project Overview:
This project predicts the quality of wine using Machine Learning and provides a simple desktop application built with Tkinter. Users can enter the wine's chemical propertie through the GUI , and the trained model predicts the wine quality instantly.

## Features:
- user-friendly Tkinter GUI
- predicts wine quality using a trained ML model.
- fast and accurate predictions
- simple input form for wine features
- easy to use desktop application.
